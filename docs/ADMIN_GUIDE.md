# ADMIN GUIDE

Product: Flowza v1.0  
Version: Flowza v1.0.1

## Purpose

Admin accounts operate content workflows for their own business scope and may optionally manage editor collaboration.

## Current Admin Capabilities (Implemented)

- Manage destinations (add, list, set default, remove)
- Compose drafts (text, photo, GIF, video, document, album)
- Publish to destination
- Create and manage schedules
- Toggle approval workflow in settings

## Current Admin Workflow

1. Open dashboard and enter Destination Manager.
2. Add destination by forwarding a channel message.
3. Create a draft from Post Composer.
4. Publish immediately or schedule for later.
5. Open Settings and toggle approval workflow if editor review is required.

## Approval Workflow Behavior (Current)

- Approval OFF: editors in that admin scope can publish.
- Approval ON: editors are blocked from direct publish.

## Planned Admin Capabilities

- Invite and manage editors via in-app UI
- Workspace-based operations
- Advanced analytics and templates UI
- Subscription management screens
