# Flowza User Guide (RC)

## 1. Getting Started

1. Open the bot and tap Start.
2. Complete setup wizard:
- Create Workspace
- Add Destination
- Create First Post
- Publish

## 2. Dashboard Navigation

Flowza is button-first. Use dashboard buttons for all module navigation.

All screens include:
- Back
- Dashboard

Typing is required only for:
- Workspace names
- Collection names
- Post content/captions
- Search queries
- Template variables
- Payment transaction hashes

## 3. Workspace Module

Use Workspace Manager to:
- Create workspace
- List/switch workspace
- Manage collections
- Manage media library
- Manage templates

## 4. Destination Module

Add destination by forwarding a message from your channel.

Flowza validates:
- Channel identity
- Bot admin permission
- Bot post permission

Use destination actions to:
- View list
- Set default destination
- Remove destination

## 5. Posts

Composer supports:
- Text
- Photo
- Video
- GIF/Animation
- Document
- Album
- Audio
- Voice

Sticker uploads are rejected with a friendly message.

Each upload creates a new draft.

## 6. Scheduler

Scheduler only schedules existing drafts.

Flow:
- Schedule
- Date
- Time
- Type
- Destination selection
- Confirm

Supported controls:
- Pause
- Resume
- Edit
- Delete

## 7. Subscription

Flowza Pro page includes:
- Current plan
- Expiry
- Remaining days
- Plans: 28/84/365 days
- TRC20 wallet and payment actions
