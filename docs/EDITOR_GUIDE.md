# EDITOR GUIDE

Product: Flowza v1.0  
Version: Flowza v1.0.1

## Purpose

Editors support content production for an Admin scope.

## Current Editor Capabilities (Implemented)

- Access post composer
- Create and edit drafts
- Access scheduler flows

## Current Editor Restrictions (Implemented)

- Cannot manage destinations
- Cannot toggle admin settings
- Publish is blocked when admin approval workflow is enabled

## Planned Editor Enhancements

- Assigned destination and workspace visibility
- Approval queue with admin feedback
- Editor activity timeline
